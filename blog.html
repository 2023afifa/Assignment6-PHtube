Q: Discuss the scope of var, let, and const

Ans: var - The scope of the var keyword is the global scope. Variables declared outside the function can be accessed globally, and variables declared in a particular function can be accessed only that function.
let - The scope of a let variable is block scoped. It can't be accessed outside the particular block. We can reassign it.
const - const keyword is also block scoped. It is same as let but we can't reassign the const variable.
